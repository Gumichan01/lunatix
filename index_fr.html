LunatiX

LunatiX est une bibliothèque de création de jeu basé sur SDL2. Il peut être utilisé pour faire un projet libre ou commercial grâce à la licence zlib/libpng.

Cette bibliothèque contient différents modules :

  • LX_Graphics : Un module d'affichage 2D reposant sur SDL_Image 2.0
  • LX_TrueTypeFont: : Un module permettant d'afficher du texte. Il est basé sur SDL_TTF 2.0
  • LX_Mixer: Module audio pour la lecture et les effet sonores
  • LX_Physics: Primitives de fonction pour la detection de collision et le déplacement d'objets
  • Et bien d'autres...

La bibliothèque fonctionne sous Windows et Linux peut marcher Mac OS X, mais pas tester).

Fonctionnalités

  • Animation 2D
  • Accélération matérielle
  • Support d'OpenGL 3.0+
  • Fenêtrage multiple
  • Système de particule
  • Multithreading
  • Saisie textuel Unicode (UTF-8)
  • Many other things...
