Detailed Description

The LX_Window class.

This class describes the window.

Note
By default, the constructor of the LX _window class gets the different informations from the LX_Configuration class
Warning
The LX_Window class must be defined only after you initialized the LX_engine (calling LX_Init())
A LX_WindowException will be occured if the window creation fails
