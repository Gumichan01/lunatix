bool LX_Graphics::LX_Image::bind ( float *  iw = nullptr,
float *  ih = nullptr 
)

Bind an image (its internal texture) to the OpenGL context of the OpenGl window where the image is drawn on in order to use the OpenGL functions.

Parameters
[out]iw(Optional) a pointer to a float value of which will be filled with the texture width. See notes.
[out]ih(Optional) a pointer to a float value which will be filled with the texture width. See notes.
Returns
TRUE on success. FALSE if the operation is not supported.
Note
1 - If provided, iw and ih will be filled with the width and height values suitable for the provided texture. In most cases, both will be 1.0, however, on systems that support the GL_ARB_texture_rectangle extension, these values will actually be the pixel width and height used to create the texture, so this factor needs to be taken into account when providing texture coordinates to OpenGL.
2 - This functions change the focused OpenGL window in order to bind the image to the OpenGL context of the window where it is usually drawn on. That is to say the function can be only called if the window is an OpenGL window. Otherwise, bind() returns FALSE.
