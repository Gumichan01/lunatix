Detailed Description

template<typename T>
class LX_Multithreading::LX_Channel< T >

Multithread synchronization primitive.

  • T Generic type that implements '='

This is a C++ implementation of the channel introduced by the Go language.

See this link:

  • https://en.wikipedia.org/wiki/Go_(programming_language)#Concurrency:_goroutines_and_channels
