void LX_Device::LX_Gamepad::close ( )

Close the gamepad

Note
An opened gamepad contains resources that are dynamiccally allocated. So, it is very important to close its resources.
Warning
If the instance is destroyed without closing the gamepad, a memory leak will happen.
