const UTF8string LX_SystemInfo::getPlatform ( )

Return the name of the patform.

Returns
The name of the platform the program is running on. Or a string beginning with "Unkownn" if the name is not available.
Note
The names returned by the function is one of these following names :
  • Windows
  • Mac OS X
  • Linux
