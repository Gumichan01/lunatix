void LX_Mixer::LX_Music::fadeIn ( int  ms)

Fade in the current music over some milliseconds of time

Parameters
[in]msMilliseconds for the fade-in effect to complete
Note
fadeIn starts playing the music with the fade-in effect. It is not necessary to call LX_Music::play() if this function is called
Any previous music will be halted, or if it is fading out it will wait (blocking) for the fade to complete

Parameters

Returns

If loops is set to LX_MIXER_LOOP, the music is played forever.
