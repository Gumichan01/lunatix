Detailed Description

The window.

Warning
The LX_Window class must be defined only after the initialization of the library (calling LX_Init())
A LX_WindowException may be occured if the window creation fails

void LX_Win::LX_Window::setDrawBlendMode ( LX_BlendMode  mode)

Set the blend mode for drawing operations (Fill, Line)

Parameters
[in]modeThe blend mode to use for blending:
Value Meaning
LX_BLENDMODE_NONE no blending
destRGBA = srcRGBA
LX_BLENDMODE_BLEND alpha blending
destRGB = (srcRGB * srcA) + (destRGB * (1-srcA))
destA = srcA + (destA * (1-srcA))
LX_BLENDMODE_ADD additive blending
destRGB = (srcRGB * srcA) + destRGB
destA = destA
LX_BLENDMODE_MOD colour modulate
destRGB = srcRGB * destRGB
destA = destA
