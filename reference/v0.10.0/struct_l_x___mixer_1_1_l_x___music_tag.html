Detailed Description

High-level music metadata.

MusicTag is a metadata structure that contains essential information about a music (title, author, album, year, ...)

In comparison with libtagpp::Tag, MusicTag does not contain low-level contents (track-gain, album-gain, bitrate, ...), but have an image loaded from the audio file if it exists.
