Detailed Description

The audio namespace.

It handles sound mixing and effect processing

Warning
In order to use this namespace, the audio flag in the configuration file must be set to 1, otherwise the behaviour of the library is undefined.

§ setPanning()

void LX_Mixer::setPanning ( uint8_t  left,
uint8_t  right 
)

Set the panning, increasing of decreasing the volume on the left or the right

Parameters
[in]leftThe volume of the left audio channel (0 - 255)
[in]rightThe volume of the right audio channel (0 - 255)
Note
This function only works on stereo audio. So the call will have no effect or fail if it is done on mono audio.
The easiest way to do true panning is to call setPanning(left, 254 - left), so that the total volume is correct, if you consider the maximum volume to be 127 per channel for center, or 254 max for left, this works, but about halves the effective volume.
To unregister this effect, use this function with 255 as left and right value or simply use LX_Mixer::removePanning().
