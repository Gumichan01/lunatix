Detailed Description

The device namespace.

This namespaces handles several types of device (gamepad, mouse) and the haptic system

Warning
In order to use any gamepad in LX_Device, the gamepad flag in the configuration file must be set to 1, otherwise the behaviour of the library is undefined.

int LX_Device::mouseCursorDisplay ( int  toggle)

Define if the cursor will be shown or not

Parameters
[in]toggleOne of these following values :
  • LX_MOUSE_SHOW: to show it
  • LX_MOUSE_HIDE: to hide
  • LX_MOUSE_QUERY: to get the current state
Returns
LX_MOUSE_SHOW if the cursor is shown, LX_MOUSE_HIDE if it is hidden, a negative value on failure.
