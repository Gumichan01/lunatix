§ LX_Colour

LX_Colour

Colour representation.

Internal structure:

uint8_t r; uint8_t g; uint8_t b; uint8_t a; // 255: opaque; 0: transparent

Examples:

Orange (decimal value) 

R,G,B: 255,165,0
⇒ LX_Colour: { r = 255, g = 165, b = 0, a = 255 }

Orange (hexadecimal value) 

#FFA500
⇒ LX_Colour: { r = 0xFF, g = 0xA5, b = 0x00, a = 0xFF }
