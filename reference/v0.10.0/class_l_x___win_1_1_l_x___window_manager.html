Detailed Description

The windows manager.

LX_WindowManager allows the user to handle several windows inside an application.

LX_Window * LX_Win::LX_WindowManager::removeWindow ( const uint32_t  id)

Delete a window from the list acording to its ID and returns the pointer to it.

Parameters
[in]idThe ID of the window that must be deleted
Returns
A valid pointer to a window if the ID refers to a valid window, a null pointer otherwise
Note
This function does not destroy the window, because each instance of a window in the list can be statically or dymanically allocated So, the user is responsible of releasing the window.
