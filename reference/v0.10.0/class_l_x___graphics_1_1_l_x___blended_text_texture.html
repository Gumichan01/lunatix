Detailed Description

The blended text texture.

This class describes a blended text texture. A blended text texture is a text that is "very" slowly but very nicely drawn on the screen.

Using this class for drawing text should be very slow, in particular if you call LX_BlendedImage::setText() and LX_BlendedImage::setSize() frequently.
