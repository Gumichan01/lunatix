bool LX_Mixer::LX_Chunk::play ( int  channel,

)

Play the current sample

Parameters
[in]channelThe channel to play the sample on
[in]loopsnumber of loops
Returns
TRUE on success, FALSE otherwise
Note
1 — The sample is played loops +1 time For example:
  • If loop == 0 → play the sample once
  • If loop == 1 → play the sample 2 times, ...
  • If loop == n (n ∈ N) → play the sample n+1 times
2 — If loops == -1 → loop forever
