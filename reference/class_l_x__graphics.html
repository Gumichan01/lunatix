Detailed Description

The LunatiX_engine graphic module.

This class describes the LX_graphics engine.

Note
: The LX_graphics class uses an instance of LX_window to get the screen
Warning
Because of the LX_window use, LX_graphics class must be built only after you initialized the LX_engine (calling LX_Init())

This function loads an SDL_Surface from any image file an optimized its format

Parameters

Returns
the loaded and optimized surface if there is no problem, NULL otherwise
Note
