SDL_Surface * LX_graphics::load_image ( std::string  filename)
static

This function loads an SDL_Surface from any image file an optimized its format

Parameters
filename: the file name which describe the image
Returns
the surface loaded and optimized, NULL otherwise
Note
You may use this function instead of load_BMP to load a non BMP surface.
Warning
1. When you call this function, the format optimization includes the alpha canal. No transparency can be put on the surface after that
2. If you want to load a bmp file and use the transparency, you should call load_BMP() instead of it and put_transparency()
