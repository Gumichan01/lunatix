Detailed Description

The LunatiX_engine True type Font (TTF) module.

This class describes the LX_ttf library. It manages the True type Font.

Note
This class supports UTF-8

Note

If you do not need to specify the font color, you may put NULL instead of this color
If the size value is 0, size get the default value, defined by DEFAULT_FONT_SIZE
Warning
If you put NULL in window, a LX_FONT_SCREEN_ERROR exception will be occured
