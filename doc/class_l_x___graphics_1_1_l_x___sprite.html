Parameters

[in]angleThe angle to rotate the sprite (in radian)
[in]mirrorThe flag that set the mirror effect:
  • LX_MIRROR_NONE: No mirror effect
  • LX_MIRROR_HORIZONTAL: Horizontal mirror
  • LX_MIRROR_VERTICAL: Vertical mirror
Note
The window is specified at object construction
