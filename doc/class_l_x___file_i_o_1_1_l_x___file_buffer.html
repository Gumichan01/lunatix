Detailed Description

The file buffer class.

This class contains information about a file buffer. It stores a memory copy of a file content into a buffer.

Constructor.

Read the file given in argument and put it in the buffer

Parameters
[in]filenameThe file to read
[in]offsetThe position in the file to start reading (default value: 0)
[in]szThe number of bytes to put in the buffer (default value: 0)
Note
1 — By default, the entire file is read from the beginning.
2 — If sz if 0, the file is read from offset to the end
Warning
If the value of offset is greater than the size of the file, then an IOException will be thrown
Exceptions
std::logic_errorIf the filename is not defined
IOExceptionIf the file cannot be read by the instance
